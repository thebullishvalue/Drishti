DRISHTI (दृष्टि) - Deep Feature MatrixA Hemrek Capital ProductDrishti is an advanced mathematical and physics-based correlation analysis engine. It moves beyond standard linear statistics to identify the absolute predictive utility of your dataset's features using Information Theory, Energy Statistics, and Network Topology.Designed for quantitative researchers and data scientists, Drishti tells you exactly which variables to keep and which to drop before training downstream Machine Learning models.🌌 Core EnginesInformation Theory (Mutual Information): Uses Shannon Entropy & Mutual Information to measure the exact "bits" of knowledge a feature provides about your target variable, capturing both linear and highly complex non-linear dependencies.Energy Statistics (Distance Correlation): Calculates Brownian Distance Correlation ($dCor$). Unlike Pearson correlation, a $dCor$ of 0 guarantees true mathematical independence between variables.Network Topology (VIF): Maps multicollinearity using Variance Inflation Factors (VIF) to detect toxic redundancy and data leakage across your feature space.Composite Oracle: Synthesizes linear stats, non-linear stats, and tree-based feature importance (Random Forest) into a unified 0-100 utility score, penalized heavily for redundancy.🚀 InstallationEnsure you have Python 3.10+ installed.# Clone the repository
